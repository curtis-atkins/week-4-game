Crystal Collector

Observe the number in the GREEN SQUARE at the start of the game. This is the user's target score.


Observe the crystal images below. The value of each crystal is unknown until the user clicks it. By clicking on any one of the four crystals the user will increase the amount of points to your current score. If the user clicks a specific crystal more than once, the same amount of points will be added to the user's current score


WIN the game by matching the user current score to the user target score. LOSE the game if the user current score goes above the user target score.


Each time the game starts, the target score as well as each crystal will be reset to a random new number.
